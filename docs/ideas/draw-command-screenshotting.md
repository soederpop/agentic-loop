# Draw Command Screenshotting

The project comes with a draw command, which just matches on the word `draw`

```shell
luca yo friday lets draw something
```

Luca's `windowManager` feature lets you screengrab or record videos so you could in theory see what is in the drawing and save it as an image, and feed that drawing back into the loop

This could be triggered by another command, which matched on `save` and `drawing`

```shell
luca yo friday save my drawing
```

we can have a docs/diagrams for images and stuff.
