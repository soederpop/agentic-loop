# Assistant Designer

A live workbench for designing and testing AI assistants.

## What it does

- Edit system prompts with live hot-reload
- Define tools with JSON Schema inputs, descriptions, and mock implementations
- Chat with your assistant and see tool calls in real-time
- Sample individual tool calls with custom inputs
- REPL for evaluating expressions against the assistant state
- Export assistant config as JSON

## When to use

- User says "I want to design an assistant" or "assistant designer"
- User wants to prototype tool schemas and system prompts
- User wants to test how an assistant responds before wiring it up
