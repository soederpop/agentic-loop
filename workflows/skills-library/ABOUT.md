# Skills Library

A browsing interface for the skillsLibrary feature. Discover skills across all configured locations, read their SKILL.md content and associated reference files, search by keyword, and scaffold new skills.

## When to use

- Browsing what skills are available across local and project directories
- Reading a skill's documentation and reference materials
- Creating a new skill from a template
- Adding new skill search locations
- Exploring community skill repositories online

## Trigger signals

- "What skills do I have?"
- "Show me the skills library"
- "I want to create a new skill"
- "Browse skills"

## When NOT to use

- If the user wants to use a skill in a conversation — the assistant's skill routing handles that automatically
- If the user wants to edit a skill's content directly — use an editor or the assistant-designer workflow
