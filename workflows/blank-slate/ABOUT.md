# Blank Slate

The first-run experience. Walks the user through establishing their foundation: a vision statement, one or more goals, and their first ideas. Designed for someone starting fresh with nothing defined yet.

## When to use

Use this when the user has no vision, no goals, or no ideas — or when they want to start over from scratch. This is the "I just got here, what do I do?" workflow.

## Trigger signals

- "I'm new here"
- "How do I get started?"
- "I don't have any goals yet"
- "I need to set up my vision"
- "Let's start from scratch"
- "What should I do first?"
- The system detects that VISION.md is unmodified, there are zero goals, or zero ideas
- The user seems lost or doesn't know what the system can do for them

## When NOT to use

- If the user already has a vision, goals, and ideas — they've graduated from this workflow
- If the user wants to check on system dependencies — use **System Setup** instead
- If the user wants to capture a single new idea — use **Capture** instead
