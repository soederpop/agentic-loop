# Assistant Gallery

A playful carousel-style showcase of all Agentic Loop assistants. Browse each assistant as a character — see their personality, voice style, and capabilities. Includes a placeholder for AI-generated character portraits.

## Features

- Carousel navigation between assistants
- Character card with portrait placeholder, ABOUT.md bio, voice personality, and catchphrases
- Voice chat button for direct conversation
- "Open Chat" button to spawn a full web-chat window (like Rocket's tools)
- Real-time voice visualization when chatting

## When to Use

When you want to browse available assistants, learn about their capabilities, or start a conversation with one.

## When NOT to Use

When you already know which assistant you want — just talk to them directly via voice wake word or use Rocket to launch a chat.
