# Ideas

A dedicated browser for viewing all of your ideas. Filter by status, goal, or tags. Click into any idea to read its full content. Designed for reviewing what you've captured so far and deciding what to work on next.

## When to use

Use this when the user wants to see their ideas, browse through them, search or filter, or read the full content of a specific idea.

## Trigger signals

- "Show me my ideas"
- "What ideas do I have?"
- "Let me look at my ideas"
- "Browse ideas"
- "What's in exploring status?"
- "Show me ideas tagged with..."

## When NOT to use

- If the user wants a high-level status overview of everything — use **Review** instead
- If the user wants to capture a new idea — use **Capture** instead
- If the user wants to develop/flesh out an idea — use **Shape** instead
