# Playbook Designer

A read-only dashboard that shows every play in the agentic loop as a card on a timeline, grouped by schedule frequency. See what's running, what ran recently, what errored, and when each play is estimated to run next.

## When to use

Use this when you want to understand the current state of the agentic loop's playbook — which plays exist, how they're scheduled, and what's been happening.

## Trigger signals

- "What plays do I have?"
- "Show me the playbook"
- "What's the agentic loop doing?"
- "When did X last run?"
- "What's scheduled?"

## When NOT to use

- If the user wants to run a specific play manually — use the task scheduler directly
- If the user wants to edit play files — use an editor (Phase 2 will add this capability)
