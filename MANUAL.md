# Agentic Loop User Manual

The Agentic loop is made up of two primary components:

- [The Engine](./docs/guides/running-the-agentic-loop.md)
- And the Fuel: ([Your Ideas](./docs/ideas/)) which are aligned to your [Vision](./docs/VISION.md) and [Your Goals](./docs/goals/)

If you run the engine, and supply the fuel, this system can grow to power everything you do on a computer, and to produce any kind of software you can dream of.

## Guides

- [Running the Agentic Loop](./docs/guides/running-the-agentic-loop.md)
- [Creating your own Assistant](./docs/guides/creating-your-own-assistant.md) not necessary, but Assistants can be used to call any code function you can write with Luca which is literally anything
- [Extending the Agentic Loop](./docs/guides/extending-the-agentic-loop.md) to add your own components to the architecture.  Maybe develop a communications service that allows you to record ideas from telegram, iMessage, email?
